datalist:

select Vs datalist
select 태그의 경우 id만 호출해서 다른 영역에서 사용하는 개념이 아니라
필요할때마다 select태그를 만들어서 작업해야 함

datalist의 경우 한군데다가 만들어 놓고 id명을 input에 호출하여
사용하는 것이 가능함!

datalist의 경우 input 태그에 value값이 한번 인식이 되버리면 그다음 항목들을
선택할수가 없다.
select의 경우엔 얼마든지 하위요소들의 값들을 변경해서 선택가능!
