transform : 객체 또는 오브젝트의 변형

transform(이동) == transform:translate(x축, y축)

transform(크기변화) == transform:scale(x축, y축)

transform(회전) == transform:rotate(45deg)

transform(x축 회전) == transform:rotatex(60deg)

transform(y축 회전) == transform:rotatey(60deg)

transform( 기울이기 ) == transform:skew(x축 각도, y축 각도)
