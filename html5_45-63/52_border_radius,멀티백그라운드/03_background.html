background

  1. background-color: 배경색상
  2. background-image: 배경이미지 호출 ( 너비, 높이 )
  3. background-repeat: 배경이미지 반복( repeat-x, repeat-y, no-repeat )
  4. background-position: 배경이미지 위치지정( top, center, bottom, left, center, right, 수치적용 가능)
  5. background-size: 배경이미지 사이즈 조절( 구형브라우저에서 적용이 안될수 있음 / 익스플로러 10버전 이상)
  6. background-attachment : 배경이미지를 해당 영역에 붙일때
