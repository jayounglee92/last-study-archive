position 속성중에서 모든 position 효과를 해제하는 기능은?

답 : static

position의 기능들 나열하고 간단하게 내용 작성하기

position:fixed(해당하는 좌표에 무조건 고정시킴, 형제 요소가 있어도 그자리로 위치함)
position:relative( 형제 요소로 사용할때: 상대는 고정 시키고 스스로의 값만 움직임 , 자신이 있던 공간을 그대로 두고 움직임
부모요소로 사용할때: absolute의 기능을 막아줄수 있음 + 상대고정역할도 유지 ) position:absolute(절대고정 : 해당하는 좌표에 고정, 절대 빈공간을 두지 않음)
position:static( 모든 포지션 기능 해제 )

position 속성들 중에서 빈공간을 두지 않고 움직일수 있는 기능은?

position:absolute

위의 기능을 적용했을때 담아줄 수 있는 position은?

부모요소에 position:relative 적용

a태그에 밑줄을 만드는 기능은?

text-decoration:underline
a태그에서만 사용하는 값이 아니라 글자가 있는 모든 대상에 사용가능

a태그에 초기화 부분 작성하기

a:link,a:visited{ text-decoration:none; color:#000;}

영역에서 넘치면 숨겨주는 값은?

overflow:hidden

글자가 넘치면 ....으로 보이게 만들기 위하여 작성하는 값들 나열하고 각각의 기능 간략히 작성하기

1) 너비와 높이 먼저 설정
2) overflow:hidden 영역에서 넘치면 숨겨주기
3) white-space:nowrap 텍스트가 한줄로 나타나도록 만들기
4) text-overflow:ellipsis 텍스트가 넘치면 ...으로 보이게 만들기

float를 끊는 방법중 가상객체를 사용하는 방법은?

가상객체사용하기 : 대상:after
content:"" : 내용이 없음
display:block : 박스로 만들기 위해 작업
clear:both ( 선택사항 상위 객체에 float가 없으면 사용하지 않아도 상관없음 )

display 기능들 아는데로 나열하고 각각의 성질들 작성하기

display:inline ( 블록성질을 인라인으로 바꿀때 )
display:block ( 인라인 속성을 블록으로도 바꾸지만, none 사라진 값을 나타나게 할때도 사용)
display:none( 화면에서 사라지게 할때 사용, 읽어주는 태그에서 사용하면 문제가 발생
헤드라인, strong, em)
display:inline-block( 인라인속성과 블록속성 같이 사용 , 단점 여백이 있음)
