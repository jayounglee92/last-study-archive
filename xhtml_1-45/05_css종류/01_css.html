인라인 스타일의 적용방식의 장.단점

+ 단점: 같은 대상에 같은 값을 적용할때 복사해야 하는 값이 너무 많기 때문에 대량으로 작업하기가 매우 불편하다.

+ 장점: 전체에 효과를 적용하는 방식이 아니라 일부의 값에 특수한 효과를 적용하기 위하여 형광펜처럼 사용함

인라인요소와 블록요소의 차이

+ 인라인 요소는 너비와 높이가 글씨크기 만큼만 차지 하기 때문에 공간의 낭비가 없음
+ 블록요소는 생성될때 브라우저의 끝에서 끝까지 영역을 차지 하기 때문에 공간의 낭비가 심하다.
