id와 class

id속성은?
+ 한번만 사용해야 하는 값들에 이름을 적용할때 사용
+ 영역이 되는 곳에 이름을 작성할때 사용
+ 소스 또는 자녀요소에 id를 설정할 경우, 기능을 하는 부분인지에 따라서 작성
+ 유일한 값을 작성할때( 제품등록번호, 주민번호, 카드번호, 쿠폰번호)
+ 같은 이름을 사용할 경우 기능은 하지만, 문법에 치명적인 오류 발생

class속성은?
+ javascript나 jQuery에서 유용한 기능을 만들때 사용한다.
+ id값은 유일하게 사용할때 적용하기 때문에 제약이 있음 이러한 단점을 class속성이 보완할 수 있다.
+ 같은 이름을 사용할 수 있다.
+ 클래스명은 영역이 되는 부분에도 작성할 수 있지만...소스가 되는 부분에도 작성할 수 있다.
+ 인식률이 id나 tag보다 좋다고 볼수는 없다. 그래서 클래스명 앞에다가 태그명을 붙여서 호출하는 경우도 종종 있다.
