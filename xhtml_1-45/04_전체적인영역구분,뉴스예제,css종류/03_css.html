css를 사용하는 이유

01) 꾸며줄때 사용 : body에서 사용하는 모든 객체를 꾸며줄수 있다.
+ 태그, id명칭이 있는 값, class라는 값이 붙어있는 객체를 꾸며줄 수 있다.

02) 편집할때 사용: 사용목적에 맞춰서 해상도 별로 작업이 가능할수 있게 함

03) stylesheet에 작업하는 것을 css라고 부른다.

xhtml == css, html5 == css3 or css3.0

04) html5 == css3(미디어쿼리, 애니메이션, 동적인 기능 작성)

style(css)의 종류

01) 인라인 스타일(inline-style) : 직접 해당하는 태그에 적용하는 방식
02) 내부적용 스타일(internal-style) : style 영역을 따로 만들어서 css를 적용하는 방식
03) 외부적용 스타일(external-style) : 외부로 style영역을 분리하여 css를 적용하는 방식( 가독성을 높이고, 작업속도가 빨라진다.)

브라우저에서 읽어주는 순서

인라인 적용방식( 태그에 직접 적용하는 방식 )

01) body(구조파트)

02) css(표현파트)

03) javascript(기능파트)

04) jQuery(기능파트)_자바스크립트 소스를 읽어줄 수 있는 환경인지 확인하고 브라우저에서 인식되기 때문에 속도가 느려질수 있음

응용문제: 인라인스타일 적용방식 사용하기

현재 p태그에 글자색상 빨간색적용하기

현재 p태그에 background-color(배경색상) 노란색 적용하기

현재 p태그에 배경색상 파란색, 글자색상 노란색 적용하기

내부적용방법( internal )_style 영역 만들기

01) style영역 만들기
02) style영역에 대상호출하기
03) 불러온 대상에 꾸며주기

인라인 적용 방식 VS 내부적용 방식

인라인 적용방식 직접 태그에 적용하는 방식으로 각각의 태그에 직접 적용해야 한다.
+ 만약에 태그가 100개라면 100개에 전체적으로 한번에 적용하기가 힘들다.

인라인 적용방식은 대량으로 같은 작업을 하는것이 아니라 각각의 중요요소에 형광펜처럼 작성하기 위하여 사용( 중요요소에 특별히 사용할때 적용하는 방식)

내부적용 방식은 태그를 요청해서 한꺼번에 같은 작업을 할 수 있다.
+ 같은 작업은 물론 각각 다른 작업도 가능!
