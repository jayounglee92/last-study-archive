제이쿼리 문법중 첫번째 문법의 장단점 작성하기

장점: php와 충돌날 우려가 전혀 없음
단점: 무한 루프 현상에 빠지고, 선택자를 선택하는 속도가 느림

input 태그와 button태그의 차이점 작성하기

input 태그는 크로스브라우징시 문제가 없음( 크기 변화에 문제가 없음)
button 태그는 크로스브라우징시 문제가 생길수 있음( 높이가 늘어날 수 있음)

id 사용시 장점은?

+ 중복되지 않고 유일한 이름으로 사용
+ 영역이나 어떤 대상을 유일하게 호출할때 사용
+ script에서 접근하기가 원활함
+ 영역을 지정할때 사용(예약어) (header, section, footer...)

jQuery나 javascript에 의존도가 높은 페이지를 만들경우 생길수 있는 문제는?

+ 3차와 4차 과정이기 때문에 화면에서 읽어 오는 속도가 매우 느림
+ jQuery의 경우에는 script 소스를 불러오기 때문에 javascript 보다 느리고
그 다음에 기능까지 읽어 오기 때문에 가장 느리다.
+ javascript의 경우에는 처음부터 만들어서 사용해야 하기 때문에 작업속도가
느려지거나 못만들 수 있음
