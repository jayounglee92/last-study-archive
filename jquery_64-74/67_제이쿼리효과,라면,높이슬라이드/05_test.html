jQuery(document).ready(function(){});
제이쿼리문서가 준비되면 기능을 해주세요.
장점 : php와 충돌날 우려가 전혀없음. 문서가 준비가 되면 안정적으로 사용할 수 있다.
단점 : 문서가 준비되지 않으면 무한로딩이 될 수 있음,선택자를 선택하는 속도가 느림

input태그와 button태그의 차이점 작성하기

input태그는 크로스브라우징시 문제가 없음(크기변화에 문제가 없음)
button태그는 크로스브라우징시 문제가 생길 수 있음(높이가 늘어날 수 있음)

id 사용시 장점은?

+ 중복되지 않고 유일한 이름으로 사용
+ 영역이나 어떤 대상을 유일하게 호출
+ script에서 접근하기가 원활함
+ 영역을 지정할 때 사용(예약어)(header, section, footer...)

jQuery나 javascript에 의존도가 높은 페이지를 만들경우 생길 수 있는 문제는?

+ 3차,4차과정이기때문에 화면에서 읽어오는 속도가 매우 느림
+ jQuery의 경우에는 script소소를 불러오기때문에 javascript보다 느리고 그 다음에 기능까지 읽어오기 때문에 가장 느리다.
+ javascript의 경우에는 처음부터 만들어서 사용해야하기 때문에 작업속도가 느려지거나 못 만들 수 있음
