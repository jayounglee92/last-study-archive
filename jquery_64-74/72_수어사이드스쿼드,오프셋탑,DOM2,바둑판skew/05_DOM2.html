DOMControl:공간생성시 선택자(0,1,2,3...), 탐색선택자

DOM : Document Object Model : 문서 존재하는 모든 객체들(태그, id, class, 공란)

공간 선택자: 필수로 사용하는 선택자

  • 0) 공간의 번호는 0부터 시작
  • 1) 홀수인 것 선택: li:odd
  • 2) 짝수인 것 선택: li:even
  • 3) 첫번째인 것 선택: li:first AND li:eq(0)
  • 4) 마지막인 것 선택: li:last
  • 5) 3번째 이전요소: li:lt(2)
  • 6) 2번째 이후요소: li:gt(1)

탐색 선택자: 권장 사용하는 선택자

  • 7) DOM2에서 li요소 중 2의 배수
  • 8) DOM2에서 li요소 중 ["APPLE"]값이 포함된 값 선택
  • 9) DOM2에서 li요소 중 [span]
  • 10) DOM2에서 li요소 중 filterTest걸러내기
  • 11) DOM2에서 li요소 중 [span]이 포함된 값 선택
