animate 사용

css영역에서 animation 사용가능? : 사용하지 못함
css3영역에서 animation 사용가능? : 사용가능
animation은 컨트롤을 할 수 없음 : 단! (hover 기능으로 작동할 수 있음) 문제점 : 버튼을 가지고 조작하기는 어려움
버튼을 조작할 수 있도록 만든 것이 jQuery의 Animate 기능
jQuery의 Animate의 경우 animation을 조작해서 사용이 가능하다는 의미로 만들어지긴했으나
animation이 가지고 있는 기능들 이외에도 동적으로 움직이는 다양한 효과를 만들어서 조작할 수 있음
