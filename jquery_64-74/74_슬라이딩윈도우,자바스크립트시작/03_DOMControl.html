DOMControl : 공간 선택자 (0,1,2,3,4 )

DOM : Document Object Model : 문서상에 존재하는 모든 객체들 컨트롤 가능

공간선택자

  • 01) 공간의 번호는 0번 부터 시작
  • 02) 홀수 인것만 선택 : li:odd
  • 03) 짝수 인것만 선택 : li:even
  • 04) p태그나 div태그, a태그 등등 나열되어 있는 태그들에서도 짝수, 홀수, eq등을 사용할수 있음
  • 05) 04번의 주의할점: 독립적으로 사용가능한 태그들은 형제요소가 끊길수 있기 때문에 주의!
  • 06) 첫번째 인것만 선택 : li:first or li:eq(0)
  • 07) 마지막 인것만 선택 : li:last , 해당하는 태그의 총 갯수를 확인 할수 있음
  • 08) 3번째 이전요소 : li:lt(2)
  • 09) 4번째 이후요소 : li:gt(3)

탐색선택자

  • 10) DOM2에서 li요소중 2의 배수: li:nth-child(2n) : 1부터 세어줌
  • 11) 10)번의 주의할점: 공간을 세는 개념이 아니라 숫자를 세는 개념(공간은 0 / 숫자는 1)
  • 12) DOM2에서 li요소중 ["APPLE"] 의 값이 표현된 값 선택 : li:contains("APPLE")
  • 13) DOM2에서 li요소중 span( 해당하는 )태그만 찾기 : li:has(span)
  • 14) DOM2에서 li요소중 .filterTest 값만 골라내기 : 대상.filter(".filterTest")
  • 15) DOM2에서 li요소중 하위요소에서 a태그 찾기 : 대상.find("a")
