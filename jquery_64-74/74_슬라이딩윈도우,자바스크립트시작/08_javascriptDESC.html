javascript와 java의 차이점

java의 특징

  1. java는 컴파일이 되어진 소스만 읽는다.
  2. 컴파일: 컴퓨터가 알아들을 수 있도록 번역해주는 일
  3. 컴파일을 위해 사용하는 프로그램: eclipse
  4. 웹접근성, 웹표준과 전혀 관련이 없음
  5. 브라우저를 고려하지 않는다.
  6. 태그를 사용하지도 않음
  7. 웹과 관련이 전혀없음!
  8. 소스 공개가 필요없음!

javscript의 특징

  1. 컴파일이 되어있지 않은 소스라도 상관없이 읽을 수 있다.
  2. 브라우저에서 읽을 수 있도록 하는 것을 목적으로 한다.
  3. 웹표준, 웹접근성을 고려할 수 있다.
  4. 반드시 브라우저에서 사용이 가능해야 한다.
  5. 태그 사용도 가능해야 한다.
  6. 소스를 숨길 수가 없다.
  7. 웹에서 기능을 위해 사용하는 프로그램은 javascript
    웹과 관련없이 프로그램을 작성하는 것은 java
